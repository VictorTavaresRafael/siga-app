# SIGA - Sistema Integrado de Gestao de Academia

O SIGA e uma solucao de software para gestao inteligente de academias, com autenticacao, check-in via QR Code e gestao de treinos.

## Estrutura
- `apps/web` - frontend Angular
- `apps/api` - backend NestJS

## Documentacao
- Frontend: `apps/web/README.md`
- Backend: `apps/api/README.md`

## Deploy (Vercel)
- Frontend: deploy do projeto `apps/web` (Vercel)
- Backend: use Railway/Render e configure `FRONTEND_URLS` com o dominio da Vercel
